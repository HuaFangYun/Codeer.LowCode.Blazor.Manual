# ColorPicker


## スクリプト
| プロパティ名          | 型               | 説明                                            |
|-----------------|-----------------|-----------------------------------------------|
| Value           | string?         | Fieldの値                                       |
| Color           | string?         | Fieldの色                                       |
| BackgroundColor | string?         | Fieldの背景色                                     | 
| IsEnabled       | bool            | Fieldの有効/無効                                   |
| IsVisible       | bool            | Fieldの表示/非表示                                  |
| IsViewOnly      | bool            | Fieldの編集可/編集不可                                |
| IsModified      | bool            | Fieldが変更されたどうか                                |
